- 다음(Daum) 뉴스 (2017. 7-12) 에서 수집한 댓글 데이터 2,393,070개에 대한 6가지 감정 분석
- 인터넷 뉴스 댓글에 감정 사전을 적용하여 Ekman의 6가지 감정(‘기쁨’, ‘슬픔’, ‘혐오’, ’놀람’, ‘공포’, ‘분노’)으로 분류된 데이터셋 구축
- 댓글 감정 데이터셋을 바탕으로 CNN 기반의 감정 분석 모델을 구축하여 약 74%의 정확도를 달성
